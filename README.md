# Demo Web Shop UI Automation Project

UI automation testing project for Demo Web Shop application using Selenium WebDriver.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Gradle

## Features
- UI automated tests
- User authentication testing
- Shopping cart testing
- Product search testing
- Assertions and validations
- Browser automation

## Test Coverage
- User registration
- User login
- Product search
- Add product to cart
- Checkout flow
- Form validation

## Project Structure
- `src/test/java` — test classes
- `src/main/java` — page objects and utilities
- `build.gradle` — project dependencies

## Technologies Used
- IntelliJ IDEA
- ChromeDriver
- GitHub

## Design Patterns
- Page Object Model (POM)

## How to Run Tests

```bash
gradlew test
```
